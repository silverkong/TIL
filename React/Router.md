# React Router

### 1. Router install

`npm install react-router-dom@6`

### 2. RESTful Route

- RESTful Route는 `url 디자인 패턴`
- Detail Page는 products/:id라고 path를 적으면 파라미터 값(가변)에 따라서 동작

### 3. useParams

- :id라고 적으면 key 값 = id
- 이것으로 value를 받아볼 수 있음

### 4. useSearchParams

- 쿼리 값을 받아올 수 있음

### 5. Redirect

- 로그인 판별 포인트
- 로그인했으면 프로필로
- 아니면 로그인 페이지로
