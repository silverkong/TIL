# HTML

> Hyper Text Markup Language
>
> 웹 브라우저에서 하이퍼 텍스트 기능을 구현하는 웹 페이지 작성 언어



### 하이퍼 텍스트

문서 간에 서로 링크가 설정되어 링크 설정 부분을 클릭하면 해당 문서로 이동하는 기능



### HTML 문서의 구성

- 태그라는 명령어
- 출력하고자 하는 문서 내용



#### 태그(tag)

- HTML에서 사용하는 명령어(문자열 기호)
- 원하는 모양과 형태로 브라우저에게 명령을 내림



#### 태그 사용 형식

- 대부분 시작 태그와 종료 태그 함께 사용

- <tag>출력 내용</tag>

  - ```html
    <p>안녕하세요</p>
    ```

- <tag value = "value"></tag>

  - ```html
    <a href="a.html" target="iFarm">이동</a>
    ```

- 예외로 종료 태그 없이 혼자 사용
  - `<br>` : 줄바꿈
  - `<img src="a.jpg">` : 이미지 삽입
  - `<hr>` : 수평선 삽입

- 태그는 대소문자 구별 없이 사용



#### HEAD

- 문서의 정보를 작성하는 부분

- 문서의 제목 설정

  ```html
  <title>문서 제목</title>
  ```

- 문자 세트 설정

  ```html
  <meta charset="UTF-8">
  ```

- 외부 파일 연결
- CSS 정의
- 자바스크립트 정의
- ...
